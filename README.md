# Alex Lindholm Personal Platform

A premium founder/operator personal website for Alex Lindholm, rebuilt from an older Wix presence into a modern Next.js app.

## Tech Stack

- Next.js app router
- Tailwind CSS
- TypeScript
- SEO metadata and JSON-LD
- Responsive, mobile-first layout
- Vercel-ready structure

## Local Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Deploy To Vercel

1. Push this project to a GitHub repository.
2. In Vercel, choose **Add New Project**.
3. Import the repository.
4. Keep the default framework setting as **Next.js**.
5. Deploy.

No environment variables are required for the current static version.

## Content Notes

The current site uses local media in `public/media/` and `public/logos/`. Add future speaking, travel, project, or partner assets there and reference them through Next/Image.
